# Solder Bee

This is a bee PCB that has eyes (LEDS) that light up in the dark and vibrates like it is flying by pressing a button that activates the motor.

## BOM
- 1 Battery Holder
- 2 LEDs
- 2 Resistors
- 1 Photoresistor
- 1 Transistor (NPN)
- 1 Push Button
- 1 DC Motor
